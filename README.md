# Prototipo-site-basico-com-Bootstrap

Projeto — Hotel da Neide
Este projeto foi desenvolvido como atividade acadêmica para praticar conceitos básicos de Bootstrap, organização de páginas, menus e carrossel de imagens.

Sobre o projeto:

O site representa a página inicial de um hotel fictício chamado Hotel da Neide, contendo:

* Navbar com menus e dropdowns
* Carrossel com imagens ilustrativas do hotel
* Títulos e breve descrição
* Lista com nomes dos integrantes do grupo

O foco principal do trabalho foi explorar e aprender recursos do Bootstrap, como grid, responsividade e componentes visuais.

Observação importante

Este projeto não teve como objetivo:

* Padronizar o tamanho das imagens
* Criar um visual altamente detalhado ou profissional

A intenção foi apenas aprendizado, sem preocupação com aperfeiçoamento visual ou tratamento das mídias.

Tecnologias utilizadas

Tecnologia  | Uso                                          

 HTML5       | Estrutura do site                            
 Bootstrap 5 | Design, responsividade e componentes visuais 

Como visualizar

Abra o arquivo `home.html` em qualquer navegador.
